⋮⋮ Calendar Clock Help

Toolbar and Overlay

The toolbar popup shows the latest stored snapshot and can open Google Calendar.

Full covers Google Calendar with the large clock. Mini shows a movable compact clock. Hide removes the overlay until opened again.

Events

Timed events draw as arcs. Single-time events and timed tasks draw as circles. Items outside the current window are counted, not drawn.

Time Window

Follow Now keeps the window around the current time. Presets and Custom set fixed 12-hour windows. 24h Radial maps a full day.

Tasks

Only visible Google Tasks with a readable time are captured. Undated tasks float on today and hide when the window does not overlap today.

Refresh and Debug

Refresh re-reads visible Calendar items. Debug's safe copies exclude calendar and task content; the separately marked full diagnostics copy is private and includes it. Calendar and Tasks each keep up to the selected -item capture limit.
